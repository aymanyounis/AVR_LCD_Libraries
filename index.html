AVR_LCD_Libraries

[AUTHOR(S)]: Ayman Younis

[DESCRIPTION]: This file contains the lcd module header file, functions to deal with the lcd, initialize, send Command, display character, display string, clear screen, go to specific row and column, display string to specific row and column and convert integer to string

  • Options:
  • You can configure it as 4-bit or 8-bit,
  • Connect the data/control bits anywhere but connect the Data sequentially and the Control at the same port any pins,
  • You can connect the CTRL and Data at the same port or different port. ##
